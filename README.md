# Anytime Karate

Simple static site for Anytime Karate (lessons, events, contact).

Put images in the `images/` folder. To push this project to GitHub:

1. Create a new repository on GitHub (via the web or `gh repo create`).
2. Add the remote locally: `git remote add origin <repo-url>`
3. Push: `git push -u origin main`
